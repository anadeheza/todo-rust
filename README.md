# To do List 
Just another basic to-do list but make it ocean styled

This project was made with to learn how to use rust as a backend with a simple db

## Features
### Tasks
To add a task simply type it in the input box and either press Enter or with the "⌯⌲" button.

The tasks will be added without any order, one below the other, but don't worry, you can reorder them by dragging them up or down.

To delete a task just press the "x" button.

You can change the task title by double clicking on it, you cannot have an empty task though, so if you erase the title no changes will be made.

If yo complete a task, you can mark it as done with the checkbox :)

### Filters
The filters just filter with the parameters told (...duh), if you press "all", you'll se all tasks, if you press "active", only the undone tasks will show, and, if you press "done", only those you have completed will appear.

### Delete all
If you don't wanna do anything (or if you have completed all tasks), you can delete them all, a warning message will appear and it'll ask you to confirm if you really wanna delete the tasks, so don't worry if you press it by accident
